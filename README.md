## 📖 Описание

Random Task Generator — это графическое приложение на Python с использованием библиотеки Tkinter для генерации случайных задач. Программа помогает организовать повседневные дела, генерируя случайные задачи из разных категорий (учёба, спорт, работа), сохраняя историю выполненных задач и позволяя добавлять собственные задачи.

## 🚀 Функциональность

- ✅ Генерация случайных задач с учётом выбранного типа
- ✅ Фильтрация задач по типу (Учёба, Спорт, Работа)
- ✅ 15 предустановленных задач
- ✅ Добавление новых задач с указанием типа
- ✅ Удаление задач из общего списка
- ✅ Отметка задачи как выполненной с генерацией новой
- ✅ История всех сгенерированных задач (сохраняется в JSON)
- ✅ Просмотр деталей задачи из истории (двойной клик)
- ✅ Очистка истории
